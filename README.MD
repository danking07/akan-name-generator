# AKAN NAME GENERATOR

## DESCRIPTION

Akan Name Generator is a web application based on Ghanaian culture.
In Akan culture, every child is given a soul name based on the day of the week they were born.
This application takes your birthdate and gender as input and returns your Akan soul name.

## Setup instructions

1. Clone this repository
2. Enter your birth day, month and year
3. Select your gender
4. Click the "Find My Akan Name" button
5. Your Akan name will appear on the screen

## BDD (Behaviour Driven Development)

1.Input Birth date and Gender

    Output: Akan name based on day of birth and gender

2.  Input: Invalid date (day <1 or day >31)

    Output: Error message indicating valid month

3.  Input: Invalid month (month <0 or >12)

        Output: Error message displaying valid date
